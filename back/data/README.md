## Media local data

Ce dossier contient les assets locaux lus par le backend.

Structure recommandee:

- `posters/` pour les affiches de films
- `videos/` pour les fichiers video sources

En base, les chemins doivent rester relatifs a `DATA_DIRECTORY`.

Exemples:

- `posters/interstellar.svg`
- `videos/interstellar.mp4`
